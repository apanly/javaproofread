#java版本中文文本纠错服务
####这个服务主要是为家具声控服务的

#服务方式
* 使用socket，端口10000 与python 的socket客户端通讯


#开发语言
* java

#依赖包
* lucene-core-3.1.0.jar 
* lucene-spellchecker-3.1.0.jar

#如何使用
* java -jar proofread.jar
